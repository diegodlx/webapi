Sistema de cadastro de usuários

Este sistema realiza o login de um usuário já cadastrado ou pode-se cadastrar um novo usuário para logar. É possível entrar na tela de gestão para verificar os usuários cadastrados e também excluir algum usuário que não faça mais parte do negócio.
